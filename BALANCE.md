# BALANCE.md

## Targets
- Case length: 30–90 sec
- Early success: ~80%

## Mechanics

Push:
+ progress
+ loop
+ discomfort

Pull:
- loop
- progress

Steer:
+ improves next move

Pressure:
- reduces loop buildup

Wash:
+ visibility

Sedation:
+ comfort (limited)

## Economy
- Base reward
- Bonuses for quality
- Penalties for failure

## Philosophy
- Tradeoffs always exist
- Speed vs quality matters
- Early game forgiving but not trivial
