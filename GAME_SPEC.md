# GAME_SPEC.md

## Game Title
"...and Around the Corner"

## Genre
Mobile 2D cartoon management / tycoon game with a procedure minigame

## Core Fantasy
You are a gastroenterologist building a practice, performing colonoscopies, hiring staff, upgrading equipment, and balancing speed, quality, and patient satisfaction.

## Tone
- Cartoon and humorous
- Light satire
- Not graphic or gross
- Clean UI

## Platform
- Mobile (iOS + Android)
- 30–90 second sessions

## Core Loops

### Procedure Loop
- Progress
- Loop Tension
- Patient Comfort
- Visibility

Actions:
- Push, Pull, Steer, Pressure, Wash, Sedate

### Day Loop
- Multiple cases
- Earn money + reputation

### Career Loop
- Hire staff
- Upgrade tools
- Improve outcomes

## Product layer

The design above is the game mechanics. The product-level strategy lives in three companion docs:

- [`AUDIENCE.md`](AUDIENCE.md) — who this game is for (medical-adjacent crowd) and the design filter that flows from it.
- [`MONETIZATION.md`](MONETIZATION.md) — niche premium at $3.99–$4.99, no ads, no IAP at launch.
- [`SIGNATURE_MOMENTS.md`](SIGNATURE_MOMENTS.md) — three deliberately shareable moments: the Cecal Photo, the Patient Archetype, the End-of-Day Stat Card.

Read these alongside this spec. Any product decision that contradicts them should either update them or be reconsidered.
