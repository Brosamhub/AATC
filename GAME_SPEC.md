# GAME_SPEC.md

## Game Title
"...and Around the Corner"

## Genre
Mobile 2D cartoon management / tycoon game with a procedure minigame

## Core Fantasy
You are a gastroenterologist building a practice, performing colonoscopies, hiring staff, upgrading equipment, and balancing speed, quality, and patient satisfaction.

## Tone
- Cartoon and humorous
- Light satire
- Not graphic or gross
- Clean UI

## Platform
- Mobile (iOS + Android)
- 30–90 second sessions

## Core Loops

### Procedure Loop
- Progress
- Loop Tension
- Patient Comfort
- Visibility

Actions:
- Push, Pull, Steer, Pressure, Wash, Sedate

### Day Loop
- Multiple cases
- Earn money + reputation

### Career Loop
- Hire staff
- Upgrade tools
- Improve outcomes
